# LeanCloud 实时通信 JavaScript SDK

## 详细使用方法请看 [官方文档](https://leancloud.cn/docs/js_realtime.html)

## 官方新版本发布流程

* 修改代码中版本号及修改日期
* 修改 bower.json 中的版本号
* 修改 changelog.md
* 提交代码合并到 master
* Github 生成 release 包
* 发布到 bower

```
bower register leancloud-realtime git@github.com:leancloud/js-realtime-sdk.git
```
